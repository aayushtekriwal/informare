# in&#8226;form&#8226;ar&#8226;e
Cycle 3 Project for 67-475.

##To Get Started:
Make sure you have PostgresSql installed. If not, get it from [here](http:postgresapp.com).

Open psql and type:
```
create database informare_development;
```

In a separate tab:
```
bundle install
rake db:migrate
rails s
```
